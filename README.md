# Language Translation Application

A modern **Language Translation Web Application** built using **React**.  
This app allows users to translate text from one language to another with a clean and responsive UI.

---

## Features

- Translate text between multiple languages
- Swap source and target languages
- Centralized state management using **Context API**
- Fast and responsive UI
- Clean layout built with **Tailwind CSS**

---

## Tech Stack

- **Frontend:** React.js
- **State Management:** Context API
- **Styling:** Tailwind CSS
- **API:** Translation API (e.g., LibreTranslate / Google Translate API)
- **Build Tool:** Vite

---
##  How It Works

1. The user enters text in the input field.
2. The user selects **source language** and **target language**.
3. When the translate button is clicked:
   - A request is sent to the translation API.
   - The translated text is returned.
4. The result is displayed in the output area.
5. Language selection and swapping are handled globally using **Context API**.
